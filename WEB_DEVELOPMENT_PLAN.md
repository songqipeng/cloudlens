## 🎯 项目概览

### 项目目标

将CloudLens CLI的核心功能扩展到Web平台，提供：
- 📊 可视化的资源管理界面
- 💰 直观的成本分析和预测
- 🔒 安全合规检查的可视化
- 🤖 批量操作和自动化能力

### 项目范围

**包含**：
- Web前端（Next.js）
- Web后端API（FastAPI）
- 核心功能模块（7个主要模块）
- UI组件库
- 文档和测试

**不包含**：
- 移动端App（后续规划）
- 多租户支持（后续规划）
- 实时协作（后续规划）

### 成功标准

1. ✅ 所有Phase 1-2功能完成并测试通过
2. ✅ 代码覆盖率 > 80%
3. ✅ 页面加载时间 < 2秒
4. ✅ 支持1000+资源展示
5. ✅ 用户满意度 > 4.5/5

---

## 🛠️ 技术栈

### 前端

- **框架**: Next.js 16 (App Router)
- **语言**: TypeScript 5
- **UI库**: React 19
- **样式**: Tailwind CSS 4
- **图表**: Recharts 3.5
- **图标**: Lucide React
- **状态管理**: React Context + Hooks（或Zustand）

- **框架**: FastAPI
- **语言**: Python 3.10+
- **ASGI服务器**: Uvicorn
- **数据验证**: Pydantic 2.0
- **WebSocket**: FastAPI WebSocket
- **测试**: pytest + httpx

### 工具

- **代码质量**: ESLint, Prettier, MyPy
- **版本控制**: Git
- **CI/CD**: GitHub Actions（可选）
- **文档**: Markdown + Sphinx（可选）
