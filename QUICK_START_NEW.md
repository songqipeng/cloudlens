# CloudLens 快速开始指南

本文档提供 CloudLens 最快速的启动方式。

---

## 🎯 选择安装模式

### 我应该使用哪种模式？

| 使用场景 | 推荐模式 | 命令 |
|---------|---------|------|
| 只想使用系统，不改代码 | 用户模式 | `curl ... \| bash` |
| 需要修改代码、二次开发 | 开发者模式 | `curl ... \| bash -s -- --dev` |

---

## 🚀 用户模式（推荐）

### 适合人群
- 只需要使用 CloudLens
- 不需要修改源代码
- 希望快速启动

### 一键安装

```bash
curl -fsSL https://raw.githubusercontent.com/songqipeng/cloudlens/main/install.sh | bash
```

### 安装内容
- ✅ Docker 环境（自动检查/安装）
- ✅ 预构建的 Docker 镜像
- ✅ 配置文件（`~/.cloudlens/`）
- ❌ 不包含源代码

### 安装时间
- **首次安装**: 3-5分钟
- **磁盘占用**: ~2GB

---

## 👨‍💻 开发者模式

### 适合人群
- 需要修改源代码
- 需要二次开发或调试

### 一键安装

```bash
curl -fsSL https://raw.githubusercontent.com/songqipeng/cloudlens/main/install.sh | bash -s -- --dev
```

### 安装内容
- ✅ 完整源代码（`~/cloudlens`）
- ✅ 开发环境（支持热重载）

### 安装时间
- **首次安装**: 5-10分钟
- **磁盘占用**: ~5GB

---

## 🆚 模式对比

| 特性 | 用户模式 | 开发者模式 |
|-----|---------|-----------|
| 安装时间 | 3-5分钟 | 5-10分钟 |
| 包含源代码 | ❌ | ✅ |
| 代码热重载 | ❌ | ✅ |
| 适合生产 | ✅ | ❌ |
| 适合开发 | ❌ | ✅ |
